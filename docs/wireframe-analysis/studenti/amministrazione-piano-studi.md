# Analisi Wireframe: Studente - Amministrazione (Piano di Studi)

**Percorso file:** `Nettuno Studenti/amministrazione/amministrazione - piano di studi.jpg` e `amministrazione - piano di studi - click su invia richiesta.jpg`

## Struttura della Pagina
Area dedicata alla visualizzazione e alla modifica del piano di studi annuale.

- **Navigazione Interna (Tabs):**
    - Carriera, Piano di studi (Attiva), Iscrizione, Tasse, Certificati.
- **Area Principale (Piano Attuale):**
    - Header con indicazione dell'anno (es. "II ANNO").
    - Tabella degli insegnamenti con colonne:
        - **Materia:** Nome del corso.
        - **Prof:** Docente titolare.
        - **CFA:** Crediti formativi.
        - **Ore:** (Visibile nel dettaglio/dopo espansione) Carico orario.
- **Barra Laterale Destra (Selezione Materie):**
    - Titolo "DA SCEGLIERE".
    - Elenco "Materia a scelta".
    - Campo di input con "+" per cercare e aggiungere una "MATERIA".
    - Azioni in basso:
        - **Annulla:** Per resettare le modifiche.
        - **Invia Richiesta:** Pulsante di conferma per sottoporre il piano alla segreteria.

## Funzionalità
1. **Visualizzazione Piano:** Consultazione degli esami previsti per l'anno in corso.
2. **Personalizzazione:** Aggiunta di materie a scelta o corsi opzionali.
3. **Sottomissione:** Invio formale del piano di studi modificato per approvazione.
4. **Validazione:** Verifica del carico di crediti (CFA) e ore totali.
