# Analisi Wireframe: Studente - Didattica (Orario Lezioni)

**Percorso file:** `Nettuno Studenti/didattica/didattica - orario lezioni.jpg`

## Struttura della Pagina
Visualizzazione temporale e logistica delle attività didattiche.

- **Navigazione Interna (Tabs):**
    - Corsi, Orario Lezioni (Attiva), Esami, Tesi.
- **Area Calendario (Sinistra):**
    - Griglia settimanale classica (Lunedì-Sabato).
    - Asse verticale per le fasce orarie.
    - Blocchi colorati/evidenziati per le lezioni programmate.
- **Area Dettaglio Giorno (Destra):**
    - Titolo "I TUOI IMPEGNI OGGI".
    - Card con info immediate: Materia, Aula, Orario.
    - **Mappa Interattiva:** Box "MAPPA" con controlli di navigazione per localizzare l'aula nell'edificio.
- **Risorse Aggiuntive (Footer Destra):**
    - Pulsante "CALENDARIO DIDATTICO PDF SCARICABILE".
    - Pulsante "SCARICA REPORT ANNUALE".

## Funzionalità
1. **Pianificazione Settimanale:** Visione d'insieme del carico didattico.
2. **Orientamento:** Supporto alla localizzazione delle aule tramite mappa.
3. **Documentazione:** Accesso a calendari istituzionali e report di frequenza.
