# Analisi Wireframe: Studente - Amministrazione (Iscrizioni)

**Percorso file:** `Nettuno Studenti/amministrazione/amministrazione - iscrizioni.jpg`

## Struttura della Pagina
Gestione delle pratiche di iscrizione, rinnovo e variazioni della carriera.

- **Navigazione Interna (Tabs):**
    - Carriera, Piano di studi, Iscrizione (Attiva), Tasse, Certificati.
- **Sezioni Principali:**
    - **Triennio:** Gruppo di azioni per studenti dei corsi triennali.
    - **Biennio:** Gruppo di azioni per studenti dei corsi biennali (Specialistica).
- **Azioni Disponibili (Card con pulsante "+"):**
    - **Rinnova:** Iscrizione all'anno accademico successivo.
    - **Cambia Indirizzo:** Richiesta di passaggio di corso o indirizzo.
    - **Sospensione:** Richiesta di interruzione temporanea degli studi.
    - **Rinuncia:** Procedura di rinuncia agli studi.
    - **Immatricolazione (solo Biennio):** Due opzioni: "Stesso indirizzo" o "Indirizzo differente".

## Funzionalità
1. **Gestione Carriera:** Avvio di procedure burocratiche complesse tramite interfaccia semplificata.
2. **Differenziazione Percorsi:** Esposizione di opzioni specifiche in base al tipo di corso (Triennio/Biennio).
3. **Workflow Guidati:** Ogni card "+" apre probabilmente un modulo specifico per completare la richiesta.
