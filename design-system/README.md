# Design System Nettuno v1.0

Benvenuto nel manuale di stile ufficiale del progetto Nettuno. Questo documento definisce le regole, i componenti e la filosofia di design per garantire coerenza su tutta la piattaforma.

## 🎯 Filosofia di Design
1.  **Universal Design**: Interfacce semplici per docenti e funzionali per studenti.
2.  **Adaptive Power**: Potenza operativa su Desktop, agilità di consultazione su Mobile.
3.  **Atomic Architecture**: Costruzione modulare (Atomi -> Molecole -> Organismi).

## 📂 Struttura del Sistema
- [**Fondamenta**](./foundations.md): Colori, Tipografia, Spazi e Griglie.
- [**Componenti**](./components.md): Bottoni, Input, Card e Modali.
- [**Iconografia**](./icons.md): Libreria di icone e regole d'uso.

## 🛠️ Stack Tecnico
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Component Library**: shadcn/ui (Radix UI)
- **Icons**: Lucide React
- **Typography**: Instrument Sans (Variable Font)
