# Iconografia

Usiamo la libreria **Lucide React**. Le icone devono essere usate con coerenza per guidare l'utente attraverso l'alfabetizzazione digitale.

## 📏 Regole d'Uso
- **Stroke Width**: `2px` (default). `2.5px` se attiva.
- **Dimensioni**:
  - `h-5 w-5`: Icone dentro bottoni o testi.
  - `h-6 w-6`: Icone di navigazione sidebar/bottom nav.
  - `h-7 w-7` o `h-10 w-10`: Icone decorative in card bento.

## 🏷️ Mappa delle Icone Principali

| Funzione | Icona | Descrizione |
| :--- | :--- | :--- |
| **Brand** | `Sparkles` | Usata per il logo principale. |
| **Home** | `Home` | Ritorno alla dashboard. |
| **Didattica** | `BookOpen` / `GraduationCap` | Libretti, corsi e tesi. |
| **Indietro** | `ArrowLeft` | Navigazione gerarchica. |
| **Avanti** | `ChevronRight` | Indicatori di click nelle liste. |
| **Notifiche** | `Bell` | Avvisi e comunicazioni. |
| **Sicurezza** | `ShieldCheck` / `Fingerprint` | Login e impostazioni privacy. |
| **Account** | `User` / `UserCog` | Area personale e impostazioni. |

## 🎨 Colori Icone
- **Default**: `text-muted-foreground` (opacity 60%).
- **Active/Focus**: `text-foreground` (opacity 100%).
- **Semantic**: `text-red-500` (errori), `text-emerald-500` (conferme).
